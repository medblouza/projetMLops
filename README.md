# Prédiction de l'Attrition Client – Projet MLOps

## Objectif
Développer un pipeline MLOps pour prédire l'attrition client à partir de données historiques clients.

## Stack MLOps
Python, Scikit-learn

DVC pour le versioning des données et du pipeline

MLflow pour le suivi des expérimentations

FastAPI pour le déploiement du modèle

Docker pour la conteneurisation

Evidently pour la surveillance en production

# data
dataset "Telco Customer Churn" (Attrition client télécoms).
